// Sobre el proyecto

SubnetLab es una aplicación web interactiva diseñada para facilitar el cálculo de subnetting y VLSM en redes IPv4. Su objetivo es ayudar a estudiantes y usuarios de redes a comprender de forma visual conceptos como dirección de red, máscara, wildcard, broadcast, hosts utilizables y división de subredes.

// Objetivo

El objetivo de SubnetLab es ofrecer una herramienta educativa, visual y práctica para el aprendizaje de redes, permitiendo realizar cálculos de subnetting y VLSM de manera clara, rápida e interactiva.
